WeatherAI

Semantic Kernel, llama3.2 ve OpenWeatherMap API kullanılarak geliştirilmiş yapay zeka destekli hava durumu asistanı.

 Özellikler

- Anlık hava durumu sorgulama
- 24 saatlik hava tahmini
- Doğal dil ile soru sorma (Türkçe destekli)
- Tamamen yerel çalışan AI model (Ollama + llama3.2)
